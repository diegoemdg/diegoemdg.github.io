# Pagina Web Personal - Diego Emilio De la Rosa Gamiño

Sitio web personal estatico creado para publicarse en GitHub Pages. Usa una paleta Mediterraneo con beige `#e9ceae`, terracota `#b75e3c` y azul petroleo `#224667`.

## Contenido

- `index.html`: pagina principal.
- `styles.css`: estilos responsivos.
- `assets/`: imagenes placeholder para reemplazar por fotos personales.
- `public_key.txt`: llave publica RSA.
- `cv-diego-emilio-de-la-rosa-gamino.pdf`: CV descargable.

## Publicacion en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir todos los archivos de esta carpeta.
3. Entrar a Settings > Pages.
4. Seleccionar la rama principal y la carpeta `/root`.
5. Guardar y esperar a que GitHub publique el sitio.

Usuario sugerido: `diegoemdg`.
